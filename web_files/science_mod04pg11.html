Find Chemical Information:
Background Information (continued)

Wikipedia can be a convenient way to find quick information. However, when you are doing academic research, you should be extremely cautious about using it as the information is not necessarily contributed by experts in the field. When you need to cite a reliable source, even for basic information, use some of the sources provided on the left.

Here are some common reliable sources you can use to find background chemical information and data.

  • The Merck Index Online is an encyclopedia of chemicals, drugs, and biologicals that contains more than 10,000 monographs. Each monograph in this authoritative reference source is a concise description of a single substance or a small group of closely related compounds.
  • The Combined Chemical Dictionary is a chemical database containing over 500,000 substances.
  • CRC Handbook of Chemistry and Physics
  • Knovel Data Search (see video tutorial)
  • For more, see the Chemistry Research Guide "Physical and Chemical Properties Data Resources"
