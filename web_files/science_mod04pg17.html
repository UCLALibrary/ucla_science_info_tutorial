Find Spectra: Web Resources

The following web resources are good starting places for searching for spectra:

  • SDBS - Spectral Database for Organic Compounds
    This free web resource has spectra for over 33,000 organic compounds, mostly FT-IR, MS, 13C-NMR and H-NMR
  • NIST Chemistry Webbook
    Free web resource with 16,000 IR spectra, 15,000 MS spectra, and 1,600 UV/Vis spectra
